## 🍭 שלב 1: החומרים הדרושים

🧪 **מצרכים בסיסיים (לכ־30 סוכריות גומי):**

| רכיב                                 | כמות            | תפקיד                          |
| ------------------------------------ | --------------- | ------------------------------ |
| מים                                  | ½ כוס           | נוזל בסיס                      |

## 🔥 שלב 2: הכנה בסיסית של תערובת גומי

1. 🥣 בקערה קטנה – ערבב את הג'לטין עם המים. תן לזה להתנפח 5 דקות.
2. 🍯 בסיר קטן – חמם את הסוכר והסירופ עד שהסוכר נמס לגמרי (בערך 110°C).
3. 🧪 הסר מהאש והוסף את תערובת הג’לטין המומסת. ערבב היטב עד שהכול נמס לגמרי.
4. 🌈 הוסף את הטעם והצבע.
5. 🐍 שפוך את התערובת לתבניות סיליקון (נחשים, דובים, חמצוצים וכו’).
6. ❄️ תן לזה להתקרר בטמפרטורת החדר שעה, ואז הכנס למקרר ל־4 שעות לפחות.

## 🍋 שלב 3: הכנת הציפוי החמצמץ

1. ערבב **2 כפות סוכר לבן + 1 כפית חומצת לימון.**
2. כשמוציאים את הסוכריות מהמקרר, גלגל כל אחת בתערובת הזו.
3. לקבלת מרקם “קראנצ’י” – אפשר לייבש בתנור על חום נמוך (40°C) ל־30 דקות.

## 🌿 טיפים וגרסאות מיוחדות

* 💧 להוסיף מעט *מיץ לימון טבעי* לתערובת כדי להעצים את הטעם.
* 🍎 אפשר לשלב **שכבות** (אדום, ירוק, צהוב) – פשוט להכין כל צבע בנפרד ולצקת בתור שכבות לתבנית.
* 🌈 לציפוי מבריק – אחרי הציפוי, לרסס במעט שמן קוקוס.
* 🌿 גרסה טבעונית: השתמש ב־**אגר אגר** במקום ג’לטין (ביחס 1:1.5) והרתח אותו במים עד שהוא מתמוסס לגמרי.
